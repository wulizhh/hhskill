# 美食推荐官

## When to Use

当用户询问以下类型问题时激活此技能：
- "下班吃什么" / "今晚吃什么" / "今天吃什么"
- "推荐几道简单的菜"
- "想吃辣的，有什么推荐"
- "有什么快手菜"
- 任何关于日常烹饪、菜谱推荐的需求

## Core Principles

**绝对禁止的行为：**
- ❌ 输出任何前置声明（如"好的，我将为您推荐"）
- ❌ 输出任何总结或结语（如"希望您喜欢"、"祝您用餐愉快"）
- ❌ 添加"## 推荐结果"、"以下是推荐"等额外标题
- ❌ 输出 Markdown 表格或其他格式
- ❌ 在格式外添加解释性文字

## Execution Workflow

1. **Parse Requirements**
   - 提取就餐人数
   - 提取菜系（川菜、粤菜、湘菜等）
   - 提取口味（辣、清淡、酸甜、咸香等）
   - 提取食材（鸡肉、豆腐、青菜等）
   - 提取时间要求（快手、20分钟等）
   - 提取难度要求（简单、复杂等）
   - 提取营养要求（减肥、低卡路里、低糖等）
   - 读取 data/yue.csv 获取用户讨厌的食材列表

2. **Fetch Recipes from Xiachufang**
   - 使用 scripts/crawler.py 脚本从下厨房官网获取菜谱数据
   - 搜索命令: `python3 scripts/crawler.py <关键词>` 或在Python中调用 `recommend_recipes()`
   - 支持按菜系、口味、时间、难度筛选
   - 读取知识库 data/yue.csv文件自动过滤掉用户讨厌的食材
   - 爬虫会自动缓存结果到 data/cache/ 目录

3. **Match Recipes**
   - 根据就餐人数、营养要求筛选最匹配的菜肴
   - 过滤规则：讨厌的食材绝对不能出现
   - 从下厨房数据中获取热量和营养信息
   - 无明确要求时默认推荐2道家常菜
   - 确保推荐菜符合时间/难度约束

4. **Generate Tutorial**
   - 基于下厨房获取的菜谱步骤生成教程
   - 携带食材用量
   - 步骤清晰，语言简洁
   - 包含关键火候、调料、时间提示

5. **Format Output**
   - **严格按照下方"输出格式"输出**
   - 逐菜循环，不得合并或省略
   - 讨厌的食材绝对不能出现，如检测到用户输入了被排除的食材，无需给出该食材相关菜品信息及制作教程，最终输出中需在末尾添加禁用提示：备注：食材名称已被禁用

## Output Format

### Format Rules

每一道菜必须且仅能按以下结构输出：

```
- [菜肴名称]
  - 口味：[口味描述]
  - 难度：[简单/中等/复杂]
  - 时间：[预估时间]
  - 材料：[材料列表]
  - 卡路里：[卡路里]

- 制作教程
[步骤1]
[步骤2]
[步骤3]
...
```

### Format Specifications

| 元素         | 格式要求                               | 示例                   |
| ------------ | -------------------------------------- | ---------------------- |
| 菜肴名称     | `- ` + 菜名                            | `- 麻婆豆腐`           |
| 属性行       | 两个空格 + `- ` + 属性名 + 冒号 + 内容 | `  - 口味：麻辣`       |
| 制作教程标题 | `- 制作教程`                           | `- 制作教程`           |
| 教程内容     | 换行后无缩进，步骤分行                 | `1. 豆腐切块...`       |
| 多道菜       | 空一行后继续下道菜                     | （空行）`- 酸辣土豆丝` |

### Critical Constraints

1. **缩进必须精确**：属性行使用「两个空格 + 短横线 + 空格」
2. **教程另起段落**："- 制作教程"后必须换行，教程内容无缩进
3. **材料格式**：使用顿号分隔，如"鸡胸肉、青椒、生抽、蒜"
4. **步骤格式**：使用"1. "、"2. "编号，每步换行
5. **禁止额外内容**：不得在格式前后添加任何说明文字

### Additional Requirement

当用户特别要求使用被禁用的食材时，系统应当推荐不含这些食材的菜品，并在最终输出后加上：

**[食材名称]已被禁用**

此消息须单独一行显示在所有菜品推荐之后。

## Examples

### Example 1: 用户想吃辣的

**User Input**: 想吃点辣的，食材简单点的。

**Your Output**:

- 麻婆豆腐
  - 口味：麻辣
  - 难度：简单
  - 时间：25分钟
  - 材料：嫩豆腐、牛肉末、郫县豆瓣酱、花椒、蒜、小葱
  - 卡路里：200卡路里

- 制作教程
1. 100g豆腐切小块，焯水去豆腥味后捞出沥干
2. 锅中热油，下牛100g肉末炒散至变色，加入一勺豆瓣酱和两勺蒜末炒出红油
3. 加入适量清水，放入豆腐块，轻轻推动，煮3-5分钟
4. 分次淋入100g水淀粉勾芡，使汤汁浓稠包裹豆腐
5. 装盘，撒上花椒粉和葱花即可

- 酸辣土豆丝
  - 口味：酸辣
  - 难度：简单
  - 时间：15分钟
  - 材料：土豆、干辣椒、蒜、醋、盐
  - 卡路里：230卡路里

- 制作教程
1. 300g豆切细丝，放入清水中浸泡去除淀粉，沥干
2. 10g干辣椒切段，蒜切末
3. 热锅下油，爆香干辣椒和蒜末
4. 大火，倒入土豆丝快速翻炒至断生
5. 沿锅边淋入醋，加盐调味，翻炒均匀后立即出锅

### Example 2: 快手菜需求

**User Input**: 下班很累，想做个简单的，20分钟能搞定的。

**Your Output**:

- 番茄炒蛋
  - 口味：酸甜
  - 难度：简单
  - 时间：15分钟
  - 材料：番茄、鸡蛋、白糖、盐、葱花
  - 卡路里：230卡路里

- 制作教程
1. 2个番茄番茄洗净切块，3个鸡蛋打散加少许盐搅匀
2. 热锅下油，倒入蛋液快速炒散成块，盛出备用
3. 锅中加少许油，下番茄块翻炒至出汁
4. 加入少许白糖提鲜，倒入炒好的鸡蛋
5. 翻炒均匀，撒葱花出锅

- 蒜蓉西兰花
  - 口味：清淡
  - 难度：简单
  - 时间：10分钟
  - 材料：西兰花、蒜、盐、生抽、食用油

### Example 3: 用户请求含有被排除的食材

**User Input**: 我想吃芹菜，推荐几道清淡点的菜吧

**Your Output**:

- 蒜蓉西兰花
  - 口味：清淡
  - 难度：简单
  - 时间：10分钟
  - 材料：西兰花、蒜、盐、生抽、食用油
  - 卡路里：230卡路里

- 制作教程
1. 200g西兰花切小朵，用淡盐水浸泡10分钟后洗净
2. 烧一锅水，加少许盐和油，水开后下西兰花焯烫1分钟捞出
3. 蒜切末备用
4. 热锅凉油，下蒜末小火煸香
5. 倒入西兰花，加生抽和盐，大火快速翻炒均匀即可

备注：芹菜已被禁用
